Change Log In Background Image on Mac OS X

December 1 2015
how to tips & tricks mac

Mac ၏ Log In screen တွင် မိမိစိတ်ကြိုက် နောက်ခံပုံများကို ပြောင်းလဲနိုင်ပါတယ်..။

၁) ပထမဦးစွာ မိမိပြောင်းလဲလိုသော Image, Photo တစ်ခုခုကို Photoshop ကဲ့သို့သော Image editor တစ်ခုခုနှင့် aspect Ratio စကေးကိုက်ညီအောင် အရင်ပြုပြင်ပါ။

အချိုးများမှာ။ ။ 1280 x 800 pixels, 1680 x 1050 pixels, 1920 x 1200 pixels တို့ဖြစ်ပါတယ်။

၂) ပြုပြင်ပြီး Image file ကို (.png) format နှင့် Desktop (သို့) တစ်နေရာရာတွင် သိမ်းပါ။ File name ကို “com.apple.desktop.admin.png” ဟု ပြောင်းပါ။

၃) ထို့နောက် “Finder” ကိုဖွင့်ပြီး “command + shift + G” ကို နှိပ်ပြီး ‘/Library/Caches’ လမ်းကြောင်းကို ရှာပေးပါ။

၄) Folder ထဲတွင်တွေ့ရသော “com.apple.desktop.admin.png” ပုံကို “com.apple.desktop.admin_backup.png” အဖြစ်ပြောင်းလဲပေးပါ။ (နောင်တစ်ချိန် ပြန်သုံးနိုင်ရန် Back Up လုပ်ထားခြင်းဖြစ်သည်)

၅) Desktop တွင် ပြုပြင်ထားသော “com.apple.desktop.admin.png” Image ကို Folder ထဲသို့ Drag & Drop ချပေးလိုက်ပါ။

၆) Log Out လုပ်ပြီး နောက်တစ်ကြိမ် Log In ပြန်ဝင်သည့်အခါ မိမိစိတ်ကြိုက် Background နှင့် Log In screen ကို မြင်တွေ့ရပါပြီ..။

အသင့်ပြုလုပ်ပြီးသား “Hope Of Myanmar” ကို အောက်ပါလင်ခ့်တွင် Download လုပ်နိုင်ပါသည်။ (အမည်ပြောင်းပြီးသား)
