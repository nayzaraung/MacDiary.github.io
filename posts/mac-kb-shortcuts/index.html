Mac Keyboard Shortcuts

December 31 2015
mac tips & tricks

        Windows ကနေ Mac ကွန်ပျူတာကို ပြောင်းသုံးကာစ လူသစ်များအတွက် ကိုင်ကာစမှာ အတော်ကသိကအောက်ဖြစ်ရတာလေးတွေ ရှိပါတယ်၊ အဲသည်အထဲမှာ Mac နဲ့ Windows ရဲ့ Keyboard layout မတူတာလဲ တစ်ခု အပါအဝင် ဖြစ်ပါတယ်။ Windows Keyboard မှာပါတဲ့ ကီးအချို့ Mac keyboard မှာ မပါရှိပါဘူး (ဥပမာ Home, PgUp, PgDn) လို ကီးမျိုးလေးတွေပေါ့။

ဒီလိုမျိုး Windows တုန်းက သုံးနေကြကီးလေးတွေရဲ့ လုပ်ဆောင်ချက်အတိုင်း Mac မှာ အသုံးပြုနိုင်တဲ့ Shortcuts လေးတွေကတော့…

“Home” နှင့် “End” Keys

        Windows မှာတုန်းက website ဖြစ်စေ တစ်ခုခုကို ဖတ်ရှုနေစဥ် စာမျက်မှာ ထိပ်ဆုံးနေရာသို့ တန်းပြီး ​ရောက်စေရန် “Home” key ကို နှိပ်ပါတယ်။ အောက်ဆုံးနေရာကို ရောက်ဖို့တော့ “End” key ကို နှိပ်ရပါတယ်။

Mac မှာ အလားတူ လုပ်ဆောင်ချက်တွေကို သုံးမယ်ဆိုရင်တော့ Function Key ဖြစ်တဲ့ “fn” နဲ့တွဲပြီး - “Home” အတွက် (fn + ◀︎ Left Arrow) နဲ့ “End” ဆိုရင်တော့ (fn + ▶︎ Right Arrow) key ကို တွဲနှိပ်ပါ။

PgUp : (fn + ▲ Up Arrow) PgDn : (fn + ▼ Down Arrow)

“Backspace” နှင့် “Delete” Keys

Mac ရဲ့ ကီးဘုတ်မှာ “Backspace” key ထည့်မထားပေးပါဘူး၊ သို့သော် ပါဝင်တဲ့ “Delete” key ကိုနှိပ်ပါက “Backspace” ရဲ့ လုပ်ဆောင်ချက်အတိုင်း cursor ရဲ့ ဘယ်ဘက်က စာများကို ဖျက်သွားပေးပါတယ်။

Windows ရဲ့ “Delete” key က cursor ရဲ့ ညာဘက်အခြမ်းက စာများကို ဖျက်ပေးပါတယ်။ Mac မှာ ထိုကဲ့သို့ အသုံးပြုနိုင်ရန် (fn + delete) keys ကို တွဲနှိပ်ပါ။

Switching Applications

Windows ကွန်ပျူတာမှာ မိမိဖွင့်ထားတဲ့ softwares/applications များအကြား အကူးအပြောင်း လုပ်နိုင်ရန် (Alt + Tab) key ကို တွဲပြီး အသုံးပြုခဲ့ကြပါတယ်။

Mac မှာ Applications တွေကြား အကူးအသန်း လုပ်လိုလျှင်တော့ (⌘ command + tab) keys ကို နှိပ်ပြီး ပြောင်းပါ။

Keyboard Shortcuts Mac အတွက် လိုအပ်သော အခြေခံ Keyboard shortcuts အချို့မှာ အောက်ပါအတိုင်း ဖြစ်ပါတယ်။

⌘ command + X : selection ပေးထားသော item နှင့် စာသားများကို Cut လုပ်ပြီး Clipboard မှာ Copy ကူးထားခြင်း။ ⌘ command + C : item, file နှင့် စာသားများကို Copy လုပ်ခြင်း၊ Finder အတွင်းမှာလည်း အလုပ်လုပ်ပါသည်။ ⌘ command + V : copy ယူထားသော အရာများကို Paste လုပ်ချခြင်း။ ⌘ command + Z : Undo ⌘ command + shift + Z : Redo ⌘ command + A : Select All

အခြား Keyboard shortcuts များကို Apple Website တွင် ဝင်ရောက်လေ့လာနိုင်ပါသည်။
