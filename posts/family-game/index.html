Classic Games Play On Mac

December 30 2015
mac how to

        ကျွန်တော်ငယ်ငယ်က Family Game တွေကို အတော်ကြီး မက်မက်မောမော ရှိခဲ့ဖူးသည်၊ ရသမျှမုန့်ဖိုး အတော်များများ ဂိမ်းဆိုင်ရောက်ကုန်တတ်၏။ တစ်ခါတစ်ရံ ဂိမ်းကစားရသည်ထက် ဂိမ်းစက် မိမိအလှည့်ရဖို့ စောင့်ရသည့်အချိန်ကပင် ပိုများပါသည်။ ထိုစဉ်က ရမ်ဘို(Contra)၊ မာရီယို(Mario Brothers)၊ တင့်ကားတိုက်ပွဲ(Battle city) အစရှိသည်တို့ကို စွဲစွဲမြဲမြဲကစားခဲ့ဖူးသည်။

နောက်ပိုင်း Nintendo, Sega, PlayStation တွေ ပေါ်လာသည့်အချိန် FIFA ဘောလုံးဂိမ်းများ ရံဖန်ရံခါဆော့ရင်း ဂိမ်းစက်များနှင့် အတော်အတန် အဆက်ပြတ်သွားခဲ့ပြီး Computer game များဘက်သို့ လှည့်သွားခဲ့သည်။ Computer တွင် ပထမဦးဆုံး စဆော့ခဲ့ဖူးသည့် ဂိမ်းမှာ PAC-Man (ထိုစဉ်က DOS ခေတ်ဖြစ်သည်) ဖြစ်ပြီး နောက်ပိုင်းမှာတော့ Prince Of Persia, Resident Evil, Dino Crisis, Age Of Empires, Need For Speed, Generals တို့မှာ အစွဲလမ်းဆုံး ကွန်ပျူတာဂိမ်းများ ဖြစ်ခဲ့သည်။ (အချို့ဂိမ်းစီဒီများ အခုထိပင် အမှတ်တရ သိမ်းထားဆဲဖြစ်သည်)

        အဲသည့်နောက် ဂိမ်းများနှင့် အ​တော်အတန်အဆက်ပြတ်သွားခဲ့ပြီး ယခု Smart Phone များပေါ်ပေါက်လာသည့် အချိန်ကျတော့မှ mobile နှင့် ဂိမ်းများပြန်လည် ကစားဖြစ်ခဲ့သည်။ တိုးတက်လာသည့် နည်းပညာကြောင့် ဂိမ်းစနစ်များ၊ ဂိမ်းရုပ်ထွက်များ အံ့မခမ်းစရာကောင်းလှပေမယ့် လူတစ်ယောက်ရဲ့ ငယ်ဘဝ စိတ်စွဲလမ်းမှုကိုတော့ အစားထိုးလို့ မရခဲ့ပါဘူး။ ယခုခေတ် ဂိမ်းတွေ ဘယ်လောက် အဆင့်မှီမှီ ငယ်ငယ်က ကစားခဲ့ဖူးတဲ့ ရိုမီယိုဘရားသား၊ ရမ်ဘိုတို့ရဲ့ Game Theme တီးလုံးလေးတွေကိုတော့ ခုထိ အလွတ်ရနေဆဲပါ..။

သက်တူရွယ်တူတွေ အနေနဲ့ ငယ်ဘဝလေးတွေကို လွမ်းလို့ပဲဖြစ်ဖြစ်၊ ဘီးကျဲကြီးတွေခေတ်က ဂိမ်းတွေကို စပ်စုချင်တဲ့ ချာတိတ်တွေပဲ ဖြစ်ဖြစ်၊ ဟိုတုန်းက ဂိမ်းပိစိလေးတွေကို Mac OS X 10.11 နဲ့ အထက်တွေမှာ ကစားကြည့်ချင်သေးတယ်ဆိုရင်တော့…။

ကစားလိုတဲ့ Game Roms များကို တော့ အောက်ပါလင့်ခ်တွင်ရှာဖွေပြီး Download လုပ်ပါ။

₪ Download ROMs

Download ရလာသော Rom ဖိုင်များကို Library ထဲသို့ Drag & Drop ချပေးပါ။

EMU Library ထဲတွင် သက်ဆိုင်ရာဂိမ်းရောက်ရှိနေပြီး Double click နှိပ်ကာ စတင်ကစားနိုင်ပါပြီ။

Game Control အတွက်တော့ “Option” icon ကို နှိပ်၍ ‘Edit Game Controls’ နဲ့ စိတ်ကြိုက် ပြောင်းလဲပါ။
