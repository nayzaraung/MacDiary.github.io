Unicode Start with 'U'...

November 27 2015
unicode on my mind

ယူနီကုဒ်စနစ် အပြောင်းအလဲကို ဦးဆောင်မည့်သူ ဘယ်သူဖြစ်မလဲ…?

မြန်မာဇော်ဂျီစာရိုက်စနစ် ကနေ မြန်မာယူနီကုဒ်စာရိုက်စနစ် ကို ပြောင်းဖို့ တိုက်တွန်းကြည့်မိတဲ့အခါ အသုံးပြုသူ(Users) အများစုက ပြောင်းလဲချင်တဲ့ ဆန္ဒရှိနေတာကို ဝမ်းမြောက်စွာ တွေ့ရှိရပါတယ်။ သို့သော် သူတို့တွင် အခက်အခဲ ကြီးြကီးမားမား (၂)ခု ရှိနေသည်က ပက်သက်ဆက်ဆံရမည့် ပတ်ဝန်းကျင်က “ဇော်ဂျီ” အများစု ဖြစ်နေခြင်းနှင့် နိစ္စဓူဝ ဝင်ရောက်ဖတ်ရှုနေသော Social media သတင်း ဝဘ်ဆိုက်အများစုက ဇော်ဂျီနှင့် ရေးသားဖော်ပြထားခြင်း စတဲ့ ရှောင်လွဲမရတဲ့ အခက်အခဲတွေပဲ ဖြစ်ပါတယ်…။

(၁)……

အချို့ Users များက “ယူနီကုဒ်စနစ်နဲ့ ပြောင်းလဲရေးကို နာမည်ကြီး လူဖတ်များ ပါဝါရှိတဲ့ သတင်းမီဒီယာ ဝဘ်ဆိုက်တွေ နဲ့ ကျော်ကြားတဲ့ အနုပညာရှင် (Celebrity) တွေကသာ စပြီးပြောင်းလဲရင် အများကပါ လိုက်ပြီး ပြောင်းလဲလာကြလိမ့်မယ်” လို့ ပြောဆိုကြတာတွေ ရှိပါတယ်…။

“မှန်သင့် သလောက်ေတာ့လဲ မှန်ပါတယ်..” ဒါပေမယ့်….

တစ်ကယ်ကတော့ သတင်း ဝဘ်ဆိုက်ေတွမှာလဲ အခက်အခဲတွေ အများကြီးရှိတယ် ဆိုတာကို ကိုယ့်တွေ့အ​နေနဲ့ ရှင်းပြပါရစေ…

ယူနီကုဒ် မဖွံ့ဖြိုးလာစဥ်ထဲက “ဇော်ဂျီဖောင့်” နဲ့ စတင်တည်ေထာင်ခဲ့တဲ့ သက်တမ်း (၆)နှစ်ခန့် ရှိသော သတင်း ဝဘ်ဆိုက်တစ်ခုမှာ ဆောင်င်းပါးပေါင်း ‘ထောင်သောင်းချီ’ ၍ ရှိနေနိုင်ပါတယ်၊ အဲဒီ ​ဆောင်းပါးများ အားလုံးကို ချက်ခြင်းတစ်ပြိုင်တည်း “ယူနီကုဒ်စနစ်” သို့ ပြန်လည်ပြောင်းလဲရမယ့် ကိစ္စမှာ လုပ်လို့ရသည် ရှိခဲ့ရင်တောင် နောက်ထပ်အခက်အခဲ တစ်ခုက ဝဘ်ဆိုက်ကို ဝင်ဖတ်မယ့် ပရိသတ် အရေအတွက် လျှော့ကျသွားနိုင်သလို စီးပွားရေးနဲ့ ဆက်စပပ်နေတဲ့ အခြားသော အကြောင်းအရာများလည်း ရှိနေနိုင်ပါသေးတယ်..။

တစ်ဖန် “ဇော်ဂျီ၊ ယူနီကုဒ်” ပြောင်းလဲရေးထက် ပိုပြီးအ​ရေးကြီးတဲ့ ပြည်သူများထံ သတင်းစီးဆင်းမှုကိုလည်း ထိခိုက်စရာ အကြောင်းတွေလည်း ရှိနိုင်ပါတယ်..။

ဒီနေရာမှာ သတင်း ဝဘ်ဆိုက်များ၊ ဘလောခ့်များ အနေနှင့် အားလုံးအဆင်ပြေအောင် လုပ်လို့ရတာ တစ်ခုရှိပါသည် “ဇော်ဂျီ” နှင့် ရေးသည်ဖြစ်စေ “ယူနီ” နှင့် ရေးသည်ဖြစ်စေ… (ရေးချင်တာနဲ့ ရေးပါ) ကိုယ့်ရဲ့ ဝဘ်ဆိုက်၊ ဘေလော့ခ်တွေမှာ “Font Embedded, EOT, Font Face” ပုံစံများနှင့် ကိုယ်အသုံးပြုမယ့် ဖောင့်ကို တစ်ခါထဲချုပ်ပြီး ထည့်ထားနိုင်ပါတယ်။ အခုလို ပြုလုပ်ထားခြင်းဖြင့် မည်သည့် မြန်မာစာဖောင့်မျှ သွင်းထားခြင်း မရှိ​တဲ့ Computers, Mobiles အားလုံးတို့မှာ အငြင်းပွားစရာမရှိပဲ (ဖောင့်မလိုပဲ) မှန်ကန်စွာ ဖတ်ရှုနိုင်ပါလိမ့်မယ်..။ (အကြံပြုခြင်းမျှသာ… ပိုမိုထိရောက် ကောင်းမွန်တဲ့ အခြားသောနည်းလမ်းများလည်း ရှိနေနိုင်ပါသေးသည်)

သို့ပေမယ့် နာမည်ကြီး ဝဘ်ဆိုက်များ၊ ဘလော့ခ်များက စတင်၍ မှန်ကန်ကောင်းမွန်သော မြန်မာယူနီကုဒ်စနစ်သို့ ဦးဆောင်ကာ ပြောင်းလဲကြလိမ့်မည်ဟု ကျွန်တော့အနေနဲ့ မယုံကြည်ပါ… သူတို့အနေနဲ့ အချိန်မရွေး ပြောင်းလဲနိုင်သော်လည်း အများပြည်သူများ၏ အရိပ်အခြေ အရွေ့ကိုသာ စောင့်ဆိုင်းနေကြပါလိမ့်မည်။ အများက ယူနီကုဒ် သုံးကြပြီဆိုတော့မှသာ သူတို့အနေနဲ့ အထက်ပါ အကြောင်းများကြောင့် လိုက်ကုဒ်နိုင်ပါလိမ့်မည်…။

“မည်သည့် မြန်မာဖောင့်မှ သွင်းထားရန် မလိုပဲ ဖတ်ရှုနိုင်သော ကျွန်တော်သိသလောက် မြန်မာဆိုက်များ”

− 7daydaily.com − burmese.dvb.no − rfa.org − bbc.com/burmese − nldchairperson.org − carsdb.com − moethagyar.blogspot.com

(၂)….

Advance User အချို့ကလည်း.. “Developers တွေကသာ ဇော်ဂျီကို (လုံးဝ) မထောက်ပံ့တော့ရင်၊ နောက်ဆုံး converter တွေကအစ မထုတ်ပေးတော့ရင် ယူနီကုဒ်စနစ်ကို လူတွေက မပြောင်းမဖြစ် ပြောင်းကို ပြောင်းလာကြမှာ..”

ဒီအတွေးမျိုး၊ ဒီစကားမျိုး ကျွန်တော်ကိုယ်တိုင်လည်း တွေးခဲ့၊ ပြောခဲ့ပါတယ်..။ ဒါပေမယ့် developers တွေ နေရာက ပြန်လည်သုံးသပ်ကြည့်တဲ့အခါမှာ၊ သူတို့က ထောက်ပံ့သူ (supporter role) ကပဲ users တွေ ဘာလိုနေလဲ လိုတာကို ဖန်တီးပေးမယ်၊ ရေးသားတီထွင်ပေးမယ် စတဲ့ users တွေရဲ့ အရိပ်အခြေကို ကြည့်ပြီး လိုအပ်တာဖြည့်ဆည်းပေးနေတဲ့ ပညာရှင်များသာ ဖြစ်ပါတယ်။ (ဦးဆောင်လှုပ်ရှားခဲ့တဲ့ ပညာရှင်အချို့ ရှိခဲ့ပေမယ့် လူအနည်းစုဖြစ်ပြီး လူမညီကြတဲ့အတွက် ထိရောက်မှု အားနည်းပါတယ်)

အခုအခါ users တွေက “ယူနီကုဒ်” ကို စိတ်ဝင်စားမှု များလာကြတဲ့အတွက် “ZawDicode” “NamKhone” အစရှိတဲ့ “ဇော်ဂျီယူနီ” နှစ်မျိုးမြင် hybrid ဖောင့်တွေ၊ Converter, Browser extension စတာတွေကို ဖန်တီးရေးသား ထုတ်ပေးလာခဲ့ပါတယ်…။ ဒါကြောင့် မြန်မာယူနီကုဒ်စနစ်သို့ Developers များက ဦးဆောင်ပြီး စတင်ပြောင်းလဲ နိုင်လိမ့်မည်ဆိုသည့် အယူအဆမှာလည်း မှားနေပြန်ပါသေးသည်။

(၃)….

သည်လိုဆိုမှဖြင့် မြန်မာယူနီကုဒ် စနစ်ကို မည်သူက ဦးဆောင်ပြောင်းလဲနိုင်ပါမည်နည်း…?

တစ်ကယ်တော့ ကျွန်တော်တို့ ခင်းဗျားတို့လို သာမန် users တွေ ကိုယ်တိုင်က စတင် ဦးဆောင်ရွေ့လျှားမှသာ ဤခရီးနီးပါတော့မည်။ အနာဂါတ်အတွက် ပြီးပြည့်စုံသော မြန်မာစာ ယူနီကုဒ်စနစ်သည် ခင်ဗျားတို့ ကျွန်တော်တို့ ၏ လက်ဖြင့် ကိုယ်တိုင် သယ်ဆောင်မှသာ အဆင့်မီသော၊ နိုင်ငံတကာမှ လက်ခံသော မြန်မာစာစနစ်တစ်ခုအဖြစ် အောင်မြင်စွာ ရပ်တည်နိုင်ပါလိမ့်မည်..။

မြန်မာယူနီကုဒ်စနစ် ပေါ်ထွန်းရေးတွင် ဦးဆောင်နိုင်လိမ့်မည်ဟု သင်တို့ ထင်ထားကြသော သူများသည် သင်တို့၏ နောက်လိုက်များသာ ဖြစ်ကြပါသည်… သင်သည်သာ ခေါင်းဆောင်နိုင်သူ ဦးဆောင်နိုင်သူ တစ်ဦးဖြစ်သည်ကို လေးလေးနက်နက် ယုံကြည်လိုက်ပါ…ဒါကြောင့် မြန်မာယူနီကုဒ်ကို သင်တို့ကစပြီး ပြောင်းလဲလိုက်ကြပါတော့..။

Unicode Start with “U”…!
